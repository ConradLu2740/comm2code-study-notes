## Phase 2：进阶 + 项目（2026.9 - 2027.2，24 周）

**核心：技术栈从基础到工业级，做 2-3 个完整项目，拿到日常实习。**

### 月计划
| 月份 | 主线技术 | 项目 | 实习 |
|---|---|---|---|
| 2026.9 | Spring Boot 基础 | 项目 1 启动：高并发秒杀 | 投日常实习 |
| 2026.10 | MySQL 深度、Redis 基础 | 项目 1 完成 v1 | 拿日常实习 offer |
| 2026.11 | Spring Cloud 入门、Nginx | 项目 1 优化 + 项目 2 启动 | 开始日常实习 |
| 2026.12 | Kafka / RabbitMQ | 项目 2：Netty IM 或分布式 RPC | 实习中 |
| 2027.1 | 系统设计基础 | 项目 2 完成 | 实习中 |
| 2027.2 | JVM 调优、面试八股 | 项目文档化、简历打磨 | 实习中 |

### Phase 2 硬指标
- ✅ 完成 2 个能写在简历上的项目（每个能讲 30 分钟，有 README + 架构图 + 压测报告）
- ✅ LeetCode 累计 **250 题**
- ✅ 3-6 个月日常实习经历（中厂起步）
- ✅ 八股文覆盖：Java 集合/并发/JVM、MySQL 索引/事务、Redis 数据结构

### 三个推荐项目

**项目 1：高并发秒杀系统**（后端经典，简历标配）
- 技术栈：Spring Boot + MySQL + Redis + RabbitMQ + Nginx + Docker
- 核心功能：登录、商品列表、秒杀下单、订单支付、限流、防超卖
- 亮点：Redis 预减库存 + Lua 脚本 + MQ 异步下单 + 布隆过滤器防穿透

**项目 2（差异化推荐）：基于 Netty 的 IM 系统** 🌟
- 技术栈：Netty + Spring Boot + WebSocket + Protobuf + Redis
- 核心功能：单聊、群聊、消息可靠投递、离线消息、心跳检测
- 亮点：自研私有协议 + 消息幂等 + 分布式集群
- **为什么推**：你的通信背景天然加分（协议设计、TCP 长连接、心跳机制），面试官会眼前一亮

**项目 3（可选加分）：短链接服务**
- 技术栈：Spring Boot + MySQL 分库分表 + Redis + ZooKeeper
- 亮点：高并发设计、分布式 ID 生成（雪花算法）

## Phase 4：暑期实习（2027.6 - 2027.8，12 周）

**核心：实习转正 or 简历镀金。**

### 实习期间目标
1. **拿到 return offer**（最理想）
2. **没转正也无所谓**：3 个月大厂经历足以让简历脱胎换骨
3. **做有产出的活**：哪怕是内部工具，能上线、有数据

### 每周必做
- 周报：本周做了什么、遇到什么问题、解决了什么
- 主动要活：不要等任务，主动找 mentor 聊
- 读团队代码：学习工程实践（CI/CD、监控、日志）
- 周末刷题 + 准备秋招（实习 ≠ 不准备秋招）

## Phase 5：秋招冲刺（2027.9 - 2027.11，12 周）🔥 关键期

**核心：拿 offer。**

### 时间线
| 时间 | 动作 |
|---|---|
| 2027.6-7 | 提前批（字节、阿里、美团最早，**7 月就开**）|
| 2027.8 | 提前批面试 + 正式批投递 |
| 2027.9 | 正式批面试密集期 |
| 2027.10 | 谈薪、签 offer |
| 2027.11 | 收尾、补录 |

### 投递策略
- 提前批能投就投（不影响正式批，多一次机会）
- 同一公司多个 BU 都投（字节最多 2 个）
- 投递顺序：**先小厂练手 → 中厂 → 大厂**

### 学历短板的应对（浙工大双非）
- 简历教育背景写"**本科东北大学（985）**"放第一行
- 突出 GPA、计算机课程成绩
- 实习经历 > 学校，1 段大厂实习能抵 985 学历
- 不投算法岗 / 数据库内核（卷到红海）
